一页速览

风格核心

Rico Reinhold 的照片最显著特点是油画般质感：低调、暖肤色、绿暗部、橙黄高光和明显颗粒共同形成厚重氛围。

主题特点

以女性人像为主，模特表现力很重要；像做菜一样，原材料、前期处理和后期调制都关键。

影调特点

整体偏暗，直方图常呈低长调，阴影像素面积多，最左侧常有缺失，所以暗部带发灰的胶片质感。

色调重点

肤色偏暖偏红，和冷白日系相反；暗部微微泛绿，高光偏橙黄色；颗粒增强画面质感。

实操顺序

参考图/裁剪 -> 基础提亮但压高光白色 -> 提阴影压黑色 -> 降清晰度提去朦胧 -> 校准阴影绿与右右左 -> 曲线抬黑压白 -> RGB S 曲线 -> HSL 红橙黄 -> 阴影绿/高光黄 -> 颗粒。

学习目标

能用主题、影调、色调三层框架拆解 Rico Reinhold 油画质感。

理解低长调不是死黑，而是暗部面积大、黑位缺失且仍有层次。

能区分 Rico 的暖肤色与日系冷白皮，并用 HSL 反向处理肤色。

掌握校准、明度曲线、RGB 曲线、HSL、颜色分级和颗粒的完整顺序。

能用绿色暗部和橙黄高光做出厚重氛围，同时避免画面脏、灰、过饱和。

主题

人物表现力、题材、姿态。

女性人像为主，模特表现力强，人物本身是画面吸引力的重要来源。

影调

直方图是否低长调，暗部是否缺失。

整体偏暗，阴影像素面积大，最左侧常缺失，暗部有发灰胶片质感。

色调

肤色、暗部、高光和颗粒。

肤色暖红，暗部微绿，高光橙黄，颗粒明显，整体像油画。

Lightroom 实操流程

参考与裁剪：打开参考视图，选择 Rico 照片作为参考；为方便演示，把素材裁成 1:1，练习时也可以按自己需要裁剪。

基础影调：素材偏暗时先提曝光；人像对比度略降，让过渡更柔；高光和白色色阶下压，避免脸部亮区过曝。

暗部层次：阴影适当提亮，让周围环境有更多细节；黑色色阶压暗，让头发暗部更实，避免画面偏灰。

柔化与平衡：降低清晰度让人像柔和；适当提高去朦胧，使画面在柔和与清楚之间取得平衡。

校准定方向：阴影色调往左偏绿，加入绿色氛围；红、绿、蓝三原色按右右左调整，让画面色调统一并更偏暖；饱和度略降，避免过艳。

明度曲线：曲线最左端上提，让直方图左侧缺失，形成胶片灰；中间点收回，避免太灰；最右侧往下压，降低亮部。

RGB 曲线：红色曲线做 S 型，让暗部偏青绿、亮部偏暖；绿色曲线做 S 型，压阴影改善发灰，提亮部；蓝色曲线压暗部、提亮部，拉开明暗色彩层次。

HSL 红橙黄：红色主要影响嘴唇：往橙、降饱和、提亮。橙色控制皮肤：往红、加饱和、提亮。黄色控制背景氛围光：往橙统一，并降低饱和和明亮度。

颜色分级：阴影加入绿色，让暗部带绿色氛围；高光加入少量黄色，不要太多；混合滑块往右，让阴影和高光色彩充分融合。

颗粒与微调：效果面板加入颗粒，并把颗粒大小和粗糙度调大一点，增强油画/胶片质感；最后可按个人喜好微调曝光或色温。

注意点

基础面板

建立低调但可读的底子

曝光上提、对比降低、高光白色压暗、阴影提亮、黑色压暗。

低调不是死黑，头发暗部要实但周围要有细节。

清晰度/去朦胧

柔化人像并保持清楚

清晰度降低，去朦胧略提高。

两者要边看边调，避免又糊又硬。

校准

建立暖调与绿暗部

阴影偏绿，三原色色相右右左，饱和略降。

校准是全局调整，幅度要轻。

明度曲线

做低长调和胶片灰

抬黑、压回中间、压亮端。

抬黑后一定要用中间点找回层次。

RGB 曲线

给暗亮部对比色

红/绿/蓝曲线做 S 型或局部压提。

RGB 曲线决定油画质感的色彩厚度。

HSL

精修嘴唇、肤色和背景黄

红往橙降饱和；橙往红加饱和；黄往橙并降饱和/明亮。

Rico 肤色偏暖红，和冷白日系相反。

颜色分级

强化暗绿高黄

阴影绿色，高光少量黄色，混合往右。

高光黄色给一点即可，过多会俗。

颗粒

增强油画/胶片质感

增加颗粒、大小和粗糙度。

颗粒明显但不能盖掉脸部细节。

练习任务

选择一张偏暗、环境光有氛围的人像素材，不要选过曝高调日系素材。

先判断主题、影调、色调：人物表现力是否强，直方图是否偏低，暗部和高光分别偏什么颜色。

用基础面板建立低调但可读的底子：提曝光、压高光白色、提阴影、压黑色。

用校准先建立暗部绿色和整体暖调，再用曲线做黑位缺失和 RGB 对比色。

用 HSL 处理嘴唇、暖红肤色和背景黄光，最后用颜色分级加入阴影绿与高光黄。

加入明显但不过量的颗粒，保存前后对比，再按个人喜好微调曝光和色温。
