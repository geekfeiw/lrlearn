一页速览

风格核心

Kai Böttcher 风格以高颜值女性人像、室内可控光、均匀曝光、轻颗粒质感和黄绿色调为核心。

主题特点

主要拍摄女性人像，模特好看是优势，但风格成立仍依赖姿态、光线、构图和后期，不是“有好看模特就行”。

影调特点

整体以标准曝光为主，不像日系小清新那样偏亮，也不像欧美电影感那样偏暗；曝光均匀、舒服，部分照片有颗粒和时尚杂志质感。

色调重点

最直观是画面带一点黄绿色。关键反直觉：阴影不是加绿，高光也不是加黄；实际是阴影轻微偏黄，高光反而偏绿。

实操顺序

裁剪贴近参考 -> 曝光/阴影/黑色上提 -> 高光略压、白色略提 -> 色温偏暖 -> 清晰度降低、鲜艳度/饱和度提高 -> 曲线抬黑提人物 -> RGB 曲线微调 -> HSL 红橙 -> 颜色分级黄/绿 -> 校准绿原色色相右移。

学习目标

能用主题、影调、色调三层框架拆解 Kai Böttcher 风格。

理解室内可控光为什么让画面更容易达到标准曝光和舒服影调。

掌握黄绿色调的真正方向：阴影和中间调偏黄，高光偏绿。

能用 RGB 曲线给画面轻微偏色，而不是用单一滑块把颜色推到夸张。

能用颜色分级和校准配合，把偏绿高光处理得更自然。

本片结论

主题

人像类型、人物比例、场景。

女性人像为主，颜值和表现力强，但成片味道来自光线、姿态、构图和后期共同作用。

影调

曝光是否偏亮/偏暗、是否有颗粒。

标准曝光、室内光线可控，整体均匀舒服；部分作品有颗粒和时尚杂志质感。

色调

黄绿效果来自哪里。

阴影轻微偏黄，高光偏绿；这点和直觉相反，是本片调色的核心。

Lightroom 实操流程

裁剪与参考先裁剪，让人物在画面中占据更大比例，也方便与参考图统一比例；打开参考视图，把 Kai 的照片放在左边。

基础提亮素材偏暗，先提高曝光，让人物脸部可见；再提亮阴影和黑色色阶，让背景暗部不再一片漆黑。

亮部与白平衡高光略微压暗，白色色阶略微提亮，让靠窗位置保留光感；色温往右偏暖，因为参考图有黄绿色氛围。

柔化与浓郁降低清晰度，利用窗边光影制造一点朦胧美；鲜艳度和饱和度都提高，让氛围更突出。

明度曲线最左端上抬，让暗部不死黑、有哑光灰质感；阴影压回，避免全图发灰；中间调和高光上提，让人物不陷入黑暗。

红色曲线参考图偏绿，红色曲线往下就是偏绿，因此整体轻微往下压。幅度必须小，否则偏色会过重。

绿色曲线绿色曲线也整体轻微往下压，让画面往暖色倾向靠近。

蓝色曲线把中间调往上提，增加一点蓝色，用来综合前面偏暖的倾向。

HSL 红橙红色主要是嘴唇：色相往橙、饱和降低、明亮度提高。橙色主要是环境氛围色：色相略往左、饱和降低、明亮度略提。

颜色分级阴影加入偏黄绿的倾向并降低明亮度；中间调加入黄色并提亮；高光加入绿色并略降明亮度。

混合、平衡与校准混合和平衡往右，让画面偏向高光绿色；如果绿色显得怪，就在校准中把绿原色色相往右拉，减弱偏绿问题并让画面更自然。

练习任务

选择一张室内或窗边人像，光线要相对可控，人物脸部不能完全死黑。

先裁剪到更接近参考图的人物比例，再打开参考视图对照。

用基础面板建立标准曝光脸部清楚、暗背景有细节、窗边有光感。

用曲线抬黑、提中间调和高光，再用 RGB 曲线轻微处理偏绿/偏暖/偏蓝关系。

用颜色分级完成阴影黄绿、中间调黄、高光绿，再通过混合和平衡偏向高光绿。

如果绿色不自然，回到校准调绿原色色相，而不是继续在颜色分级里硬推。
